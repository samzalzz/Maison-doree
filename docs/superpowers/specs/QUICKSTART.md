## 🎯 What Each Role Does

### Admin
**Full System Control**
- Create and manage labs, machines, recipes
- Schedule production batches
- Monitor inventory across all locations
- Review analytics and forecasts
- Create users and assign roles

**Access**: Admin Menu → All pages

### Manager  
**Day-to-Day Oversight**
- Monitor production progress
- Manage team assignments
- Ensure quality standards
- Review performance metrics
- Escalate issues to admin

**Access**: Production → Manager Dashboard

### Worker
**Hands-On Production**
- See assigned production tasks
- Execute recipes step-by-step
- Report batch progress and issues
- Record actual production times
- Request support if needed

**Access**: Worker Dashboard (simple, task-focused)

## 🚀 Common Tasks

### For Admins

**Create a new production batch:**
1. Go to Production → Batches
2. Click "Create New Batch"
3. Select lab, recipe, quantity
4. Click "Create"

**Check inventory:**
1. Go to Production → Stock
2. Expand a lab to see materials
3. Click edit icon to adjust quantities

**View production overview:**
1. Go to Production → Dashboard
2. See active batches, lab capacity, material alerts
3. Click to drill into details

### For Managers

**Monitor today's work:**
1. Go to Production → Manager Dashboard
2. Review task list and alerts
3. Address any issues that appeared

**Check worker performance:**
1. Go to Production → Manager Dashboard
2. Scroll to "Team Metrics"
3. Review productivity and quality ratings

**Make a quality decision:**
1. Find completed batch in Production → Batches
2. Click to review quality notes
3. Click "Approve" or "Reject"

### For Workers

**See your tasks:**
1. Log in → You see your tasks immediately
2. Expand any task to see recipe details
3. Follow the instructions

**Report completion:**
1. When finished, click "Mark Complete"
2. Enter actual time and any notes
3. Click "Submit"

## ⚡ Five-Minute Setup

**For a brand new lab:**

1. **Admin**: Go to Production Management → Labs
   - Click "Add Lab"
   - Enter name (e.g., "Mixing Lab")
   - Select type (PREPARATION/ASSEMBLY/FINISHING)
   - Click "Create"

2. **Admin**: Go to Production Management → Machines
   - Click "Add Machine"
   - Select the lab you just created
   - Enter machine name (e.g., "Oven A")
   - Set capacity (max items per batch)
   - Set cycle time (minutes to complete)
   - Click "Create"

3. **Manager**: Go to Production → Manager Dashboard
   - You'll see the new lab appear
   - It's ready for production

4. **Admin**: Go to Production → Stock
   - Find your new lab
   - Click "Add Material"
   - Select materials needed
   - Set quantities
   - Click "Add"

5. **Admin**: Create your first batch
   - Go to Production → Batches
   - Click "Create New Batch"
   - Select your lab and an existing recipe
   - Click "Create"

6. **Worker**: Log in
   - You'll see the batch assigned to your lab
   - Follow the recipe instructions
   - Mark complete when done

## 🆘 Common Issues

### "I can't create a batch"
- ✓ Check: Do we have enough materials in that lab?
- ✓ Check: Is there equipment available?
- ✓ Check: Is the lab below its capacity?

### "A batch is taking too long"
- Go to the batch details
- Check if there are reported issues
- See if additional resources can help
- Contact supervisor if you need help

### "I don't see material in the lab"
- Check Production → Stock
- Find the lab and see what's in stock
- If missing, use "Add Material" button
- Or request transfer from another lab

### "My role is wrong"
- Contact your administrator
- Go to Settings → Users
- Click on your user and update role
- Log out and back in
