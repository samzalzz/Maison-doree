## Production Tasks

### Understanding a Batch

Each batch card shows:
1. **Batch Number**: Unique identifier (BATCH-2024-04-001)
2. **Recipe Name**: What you're making (e.g., "Tarte aux Fraises")
3. **Quantity**: How many units to produce (e.g., 50 pieces)
4. **Start Time**: When to begin production
5. **Estimated Duration**: How long it should take
6. **Current Status**: 
   - 🟡 **PLANNED** - Ready to start when scheduled
   - 🟠 **IN_PROGRESS** - Currently being produced
   - ✅ **COMPLETED** - Finished production
   - ⏸️ **PAUSED** - Temporarily halted

### Viewing Recipe Details

1. Find the batch in your task list
2. Click **"View Recipe"** or expand the batch card
3. See:
   - **Ingredients List**: All materials needed
   - **Labor Instructions**: Step-by-step process
   - **Equipment Required**: Machines to use
   - **Quality Standards**: What "done" looks like
   - **Estimated Labor Time**: How long it should take

### Starting a Batch

1. When scheduled start time arrives, find the batch
2. Check that:
   - ✓ All equipment is available and working
   - ✓ All materials are in the lab
   - ✓ Team members are ready
3. Click **"Start Production"** button
4. Batch status changes to **IN_PROGRESS**
5. Begin following the recipe instructions

### During Production

- Follow recipe instructions exactly
- Use assigned equipment properly
- Maintain food safety and hygiene
- Monitor baking times and temperatures
- Call supervisor if issues arise

## Reporting Progress

### Completing a Batch

When production is finished:

1. Find the batch in your task list
2. Click **"Mark Complete"**
3. Enter completion details:
   - **Actual Completion Time**: When you finished
   - **Quality Notes**: Any issues or observations
   - **Actual Quantity**: How many units completed
   - **Material Waste**: Any loss or defects
4. Click **"Submit"**

### Progress Updates

For longer batches:
1. Click **"Update Progress"**
2. Report:
   - How many units completed so far
   - Any issues encountered
   - Est. time remaining
3. Click **"Save Update"**

## Safety & Quality

### Food Safety
- Always wear clean work clothes and hairnet
- Wash hands before handling food
- Use sanitized equipment
- Check ingredient freshness before use
- Store materials at correct temperatures

### Quality Standards

Before completing a batch, verify:
- ✓ Correct size and weight
- ✓ No visible defects
- ✓ Proper color and texture
- ✓ Correct taste/flavor
- ✓ Proper labeling/packaging

### Reporting Issues

If something goes wrong:
1. Stop production immediately
2. Click **"Report Issue"** in the batch details
3. Describe what happened
4. Supervisor will contact you
5. Don't discard materials unless instructed

## Tips for Success

### Efficiency
- Read recipe completely before starting
- Prepare all ingredients first (mise en place)
- Use batching for similar tasks
- Clean equipment between batches
- Ask for help from experienced team members

### Quality
- Measure ingredients precisely
- Don't skip quality checks
- Report defects immediately
- Follow standard procedures
- Ask questions if unsure

### Teamwork
- Communicate with your team
- Help colleagues when possible
- Share knowledge about recipes
- Report equipment problems promptly
- Celebrate batch completions

## FAQ

**Q: What if I need to pause production?**
A: Click "Pause Batch" and enter reason. Supervisor will review.

**Q: How long should this batch take?**
A: Check the recipe details - "Estimated Labor Time" shows expected duration.

**Q: Can I move a batch to another lab?**
A: No, contact supervisor if there's an issue.

**Q: What if ingredients run out mid-batch?**
A: Click "Report Issue" - supervisor can add stock from another lab.

**Q: How do I request time off?**
A: Contact your lab manager directly (outside this system).
