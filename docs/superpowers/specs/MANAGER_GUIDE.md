## Overview

As a production manager, you oversee day-to-day operations, ensure quality, and optimize lab efficiency. Your role bridges administrators (strategy) and workers (execution).

### Key Responsibilities
- Monitor production status across labs
- Manage team allocation and capacity
- Ensure quality standards are met
- Report on production metrics
- Escalate issues to administrators

## Manager Dashboard

Navigate to **Production → Manager Dashboard** to see:

### **Lab Status Overview**
For each lab, see:
- **Capacity Status**: Current utilization percentage
- **Active Batches**: Number of in-progress tasks
- **Team Count**: Employees on shift
- **Equipment Status**: Machines available/down
- **Material Status**: Low stock warnings

### **Production Timeline**
- Gantt-style view of all batches
- Planned vs. actual completion times
- Batch dependencies
- Critical path items (bottlenecks)

### **Team Metrics**
- Productivity by employee
- Batch completion rates
- Quality ratings
- Time efficiency (estimated vs. actual)

### **Alerts & Issues**
- Low inventory warnings
- Equipment maintenance needed
- Missed deadlines
- Quality concerns
- Worker absence/availability

## Team Management

### Daily Team Assignment

1. Review worker assignments for the day
2. Check availability:
   - Who's scheduled
   - Skill levels
   - Experience with recipes
3. Balance team across labs:
   - Ensure each lab has adequate coverage
   - Assign experienced staff to complex recipes
   - Cross-train junior staff
4. Report any absences:
   - Contact admin if coverage is insufficient
   - Adjust batches if needed

### Worker Monitoring

1. Access **Team Performance** section
2. View per-worker metrics:
   - Batches completed
   - Average completion time
   - Quality ratings
   - Efficiency trends
3. Identify:
   - Top performers (recognition)
   - Workers needing training (support)
   - Bottlenecks in production

### Quality Control

1. Review completed batches regularly
2. Check quality notes from workers
3. Verify against standards:
   - Correct size/weight
   - No defects
   - Proper appearance
   - Taste/texture quality
4. Record any quality issues:
   - Log defect details
   - Identify root cause
   - Recommend corrective action

## Production Oversight

### Monitoring Batch Progress

1. Go to **Production → Batches**
2. For each batch, track:
   - Status (Planned → In Progress → Completed)
   - Actual vs. planned timeline
   - Quantity progress
   - Any issues reported

3. If batch is delayed:
   - Check reported issues
   - Assess if additional resources help
   - Escalate if needed

### Handling Issues

**Equipment Problems**
- Check equipment status
- Request maintenance if broken
- Reassign batch to different machine if possible

**Material Shortage**
- Verify stock levels
- Request transfer from another lab if possible
- Escalate order to admin if needed

**Quality Issues**
- Review quality report from worker
- Determine if batch can be salvaged
- Decide whether to reject or accept
- Document for analytics

**Worker Unavailability**
- Reassign batch to available worker
- Combine smaller batches if necessary
- Request admin support if insufficient coverage

### Decision Making

**Should we start this batch?**
- ✓ Do we have all materials?
- ✓ Is equipment available?
- ✓ Do we have qualified staff?
- ✓ Is lab capacity available?

**Should we pause this batch?**
- Equipment failure (yes)
- Minor material shortage (maybe - request transfer)
- Worker unavailability (maybe - reassign if possible)
- Quality concern (yes - investigate)

**Should we reject a completed batch?**
- Significant quality defects (yes)
- Wrong quantity (yes)
- Safety concerns (yes)
- Minor cosmetic issues (probably not)

## Performance Metrics

### Key Performance Indicators (KPIs)

1. **On-Time Delivery**
   - % of batches completed on schedule
   - Target: >95%

2. **Quality Rate**
   - % of batches passing quality check
   - Target: >98%

3. **Efficiency**
   - Actual time vs. estimated time
   - Target: within 10%

4. **Capacity Utilization**
   - % of lab capacity in use
   - Target: 70-85% (higher = waste, lower = under-utilized)

5. **Material Yield**
   - % of materials converted to product
   - Target: >95%

### Reviewing Metrics

1. Go to **Core → Analytics**
2. View trend charts:
   - Production volume over time
   - Quality trends
   - Equipment efficiency
   - Material usage
3. Identify patterns:
   - Declining quality (training needed?)
   - Efficiency drops (equipment issue?)
   - Increased waste (process problem?)

### Taking Action

**If On-Time Delivery < 90%**
- Check which batches are delayed
- Identify root causes
- Increase planning buffer
- Add capacity if consistently over

**If Quality < 95%**
- Review failed batches
- Identify common issues
- Implement targeted training
- Review recipe instructions
- Consider process changes

**If Utilization < 60%**
- Underutilized capacity
- Review demand forecast
- Increase batch sizes
- Combine batches where possible
- Plan maintenance

**If Utilization > 90%**
- Over-capacity situation
- Risk of quality issues
- Worker burnout risk
- Request additional capacity
- Adjust schedule to smooth demand

## Communication with Admin

### Report Issues

Go to **Production → Manager Dashboard** and use "Report Issue" to:
- Request additional materials
- Report equipment downtime
- Alert to quality concerns
- Request capacity expansion
- Escalate worker performance issues

### Recommendations

Always explain:
- **What**: Describe the issue or opportunity
- **Why**: Impact on production or quality
- **How**: Recommended action
- **When**: Timeline for decision needed

### Regular Briefings

Prepare for admin meetings:
- Weekly production summary
- KPI trends and concerns
- Key decisions made
- Resource requests
- Upcoming challenges

## Tips for Success

1. **Stay Visible**: Be present in labs, see issues first-hand
2. **Communicate**: Keep team and admin informed
3. **Be Proactive**: Anticipate problems before they occur
4. **Support Workers**: Help solve problems, not just assign blame
5. **Document Everything**: Track issues and decisions
6. **Know Your Recipes**: Understand production processes
7. **Lead by Example**: Follow safety and quality standards
